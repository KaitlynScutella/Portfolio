Second Level Block Heading

What are block elements good for?

The next element after this is a ul, or unordered list. It makes a bulleted list. Your list can contain anything you'd like, and does not have to be in any specific order!

  • tea
  • coffee
  • milk
  • juice
  • water

How about we look at an ol element, or ordered list? The code looks very similar to the unordered list. You have to see the difference by viewing in a web browser. It is used to a list where you put things in order from first to last.

  1. breakfast
  2. lunch
  3. dinner
  4. dessert
  5. snacks
